Changelog

Every shipped change, newest first. Updates reach your Jira Cloud site automatically through Forge — there is nothing to download.

3.29.0 — Epic & parent rollup

June 2026

Parent issues now roll up automatically. Each epic (or any parent with children) draws a slim summary bar that spans from its earliest child’s start to its latest child’s end, with a duration-weighted % complete shown above it. Summary bars are read-only — they always reflect their children, so the timeline stays in sync as you reschedule the work underneath. Multi-level hierarchies (epic → story → sub-task) roll up correctly, bottom-up.

3.28.0 — Export to MS Project

June 2026

Export your Gantt — tasks and dependencies — to MS Project XML from the toolbar “…” menu, ready to open in Microsoft Project. Joins PNG, CSV and PDF export.

3.27.0 — Velocity Mode + smoother planning

June 2026

  • Velocity Mode — a new Capacity view: see each assignee's workload, exactly who's over capacity this week, and click a person to see the tasks driving their load.
  • Resizable split — drag the divider to rebalance the table and the timeline.
  • Drag to scroll — press-and-hold to pan both the timeline and the table.
  • Smoother layout — removed a stray page scrollbar, kept hover tooltips on-screen near edges, and fixed bulk +1 / -1 day moves so they save back to Jira.

3.26.0 — Export to PDF

June 2026

Export your Gantt chart straight to a print-ready PDF from the toolbar “…” menu — ideal for status reports and stakeholder updates. Joins the existing PNG and CSV exports.

3.25.0 — Export to CSV

June 2026

Export the tasks currently shown (filters respected) to CSV — Key, Summary, Type, Status, Assignee, Start, End, Duration, Progress and Parent — for spreadsheets and reporting.

3.24.0 — In-app feedback, usage signals & timeline polish

June 2026

  • Send feedback button in the toolbar — reach us by email or a prefilled GitHub issue, with an admin-only feedback inbox.
  • Anonymous usage signals to help prioritise what we build next (no PII).
  • Press-and-hold to pan the timeline horizontally.
  • Drag preview — a ghost of the card shows exactly where a bar will land while you move it.
  • Toolbar and dropdown-menu polish.

3.15.1

April 2026

Removed the in-panel Open in GanttBoard deep-link button while we complete cross-project-type navigation testing. Will return in a future release.

3.15.0 — Marketplace launch

April 2026

First public release on the Atlassian Marketplace. The full GanttBoard feature set is available from day one:

  • Full Gantt timeline with drag-to-reschedule and drag-to-resize bars
  • Spreadsheet table view with resizable, reorderable, freezable columns
  • Four dependency types (FS / SS / FF / SF) with drag-to-create and lag support
  • Critical path (CPM) highlighting with float calculation
  • Baselines — save snapshots, show drift with ghost bars
  • Auto-scheduling across the full dependency graph
  • Saved views and color rules
  • Facilitated standup mode with shuffled roster
  • MS Project XML round-trip import / export
  • PNG and CSV export of the active Gantt
  • Keyboard shortcuts — press ? for the full panel
  • Light and dark mode that follows your Jira theme
