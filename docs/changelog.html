3.15.0 — Marketplace launch

April 2026

First public release on the Atlassian Marketplace. The full GanttBoard feature set is available from day one:

  • Full Gantt timeline with drag-to-reschedule and drag-to-resize bars
  • Spreadsheet table view with resizable, reorderable, freezable columns
  • Four dependency types (FS / SS / FF / SF) with drag-to-create and lag support
  • Critical path (CPM) highlighting with float calculation
  • Baselines — save snapshots, show drift with ghost bars
  • Auto-scheduling across the full dependency graph
  • Saved views and color rules
  • Facilitated standup mode with shuffled roster
  • MS Project XML round-trip import / export
  • PNG and CSV export of the active Gantt
  • Keyboard shortcuts — press ? for the full panel
  • Light and dark mode that follows your Jira theme
