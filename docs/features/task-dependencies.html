Task dependencies & dependency arrows

A Gantt chart without dependencies is just a date picker with a background image. GanttBoard supports all four classical types — Finish-to-Start, Start-to-Start, Finish-to-Finish, and Start-to-Finish. Hover a bar, grab one of the two dots, and drag to a dot on another bar; GanttBoard infers the type from which dots you connected and lets you add lag.

  • L-shaped arrows connect linked tasks.
  • Add a lag (e.g. “start 3 days after”) right in the dialog.
  • Dependencies feed critical path and auto-scheduling.
