Color rules

Make the timeline readable in a glance by color-coding bars with rules. Color by status, assignee, issue type, or any field — for example, red for blocked, green for done, or a distinct color per team. Rules are saved with your view.

  • Match on a field and value, set bar and/or row color.
  • Stack multiple rules in priority order.
  • Saved per view, so different cuts can look different.
