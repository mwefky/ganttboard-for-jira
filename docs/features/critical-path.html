Critical path (CPM)

Once your tasks are linked, toggle Critical path from the toolbar and GanttBoard highlights the longest chain through your project — the set of tasks where any delay pushes the end date. This is the classic Critical Path Method, the same result MS Project or Primavera would produce, applied directly to your Jira issues.

  • One-click toggle in the toolbar.
  • Updates instantly as you add or change dependencies.
  • See at a glance which work is schedule-critical.
